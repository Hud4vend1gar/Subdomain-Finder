## **Installation**
`git clone https://github.com/Hud4vend1gar/Subdomain-Finder.git`

## **Dependencies**
Subdomain-Finder depends on the requests and urllib python modules.

- `pip install requests`
- `pip install urllib`

## **Usage**
Just run main.py
- `python main.py`
